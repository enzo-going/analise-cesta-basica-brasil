Análise da Cesta Básica no Brasil

Projeto de análise exploratória de dados sobre a evolução do preço da cesta básica e das horas de trabalho necessárias para sua aquisição em capitais da região Sudeste do Brasil.

O trabalho utiliza dados da Pesquisa Nacional da Cesta Básica de Alimentos, com foco em transformar dados socioeconômicos em visualizações, estatísticas descritivas e uma interface web de apoio à leitura pública dos resultados.

Objetivo

Analisar a variação histórica do custo da cesta básica e sua relação com o poder de compra do trabalhador, considerando:

evolução do preço da cesta básica;
tempo de trabalho necessário para aquisição;
comparação entre capitais do Sudeste;
impactos econômicos em períodos críticos;
apresentação dos resultados em interface web acessível.

Este projeto foi desenvolvido originalmente como parte de uma Pesquisa Curricular de Graduação, envolvendo análise de dados, Interação Humano-Computador, acessibilidade digital e publicação de informações em interface web.

Esta versão do repositório tem como objetivo reorganizar o material original em uma estrutura mais clara, reprodutível e adequada para portfólio técnico.
